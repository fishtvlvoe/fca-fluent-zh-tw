為所有 FCA 系列與 Fluent 系列外掛提供繁體中文翻譯，不修改原始外掛檔案，更新外掛不受影響。

== Description ==

此翻譯包涵蓋以下外掛的繁體中文翻譯：

**FCA 系列**
* FCA Boards
* FCA Content Manager
* FCA Course Blocks
* FCA Events
* FCA Hub
* FCA Knowledgebase
* FCA Multi Reactions
* FCA Pages
* FCA Push Notifications
* FCA PWA
* FCA Widgets
* FCE Quick Fixes
* FCE Shortcodes

**Fluent 系列**
* Fluent Booking / Pro
* Fluent Cart / Pro
* Fluent Community / Pro
* FluentCRM
* Fluent Player / Pro
* Fluent Security
* Fluent SMTP
* FluentCampaign Pro
* Fluent Forms / Pro

== Installation ==

1. 上傳 `fca-fluent-zh-tw` 資料夾到 `/wp-content/plugins/` 目錄
2. 在 WordPress 後台啟用「FCA & Fluent 繁體中文翻譯包」外掛
3. 確認 WordPress 語言設定為「繁體中文（台灣）」
